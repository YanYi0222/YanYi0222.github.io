## ✨ 網站亮點介紹

我們希望讓看數據這件事變得很有趣，就像在玩手機 App 一樣順暢：

- 🎨 **活生生的背景畫面**
  網站背景不是一張死板的圖片，而是像水滴一樣緩慢流動的光球，搭配微微透光的「毛玻璃」效果，讓畫面看起來既現代又舒服。
- 🍱 **像便當盒一樣整齊的排版**
  我們把龐雜的資訊整理成一格一格的卡片，不論您是用電腦還是手機看，畫面永遠都乾淨整齊。
- 🌓 **一鍵切換日夜模式**
  晚上看螢幕太刺眼？按右上角的月亮按鈕，整個網站（包含地圖和圖表）會瞬間變成超有質感的深色模式，保護您的眼睛。
- 🗺️ **真實的互動地圖**
  我們把捷運和火車的路線畫進了網站裡！點擊地標，地圖還會自動帶您飛過去，並秀出當地的照片與介紹。
- 📊 **一看就懂的雷達圖**
  「生活機能」、「房價親民度」等分數，全部化為六角形的雷達圖，北屯和竹東哪裡強、哪裡弱，兩張網疊在一起秒懂。
- 👤 **專屬您的客製化推薦**
  我們準備了「頂客族」、「小家庭」與「退休族」三種劇本。點選您的身分，網站就會告訴您哪一區更適合您！

## 🛠️ 怎麼做出來的？（無痛技術版）

為了讓網頁跑得最快，我們**沒有使用任何肥大、厚重的網頁產生器或複雜的框架**。

- 我們親手使用網頁最基礎的語言（HTML、CSS、JavaScript）一磚一瓦刻出所有畫面。
- 使用了開源的地圖工具（Leaflet）與圖表工具（Chart.js），讓資料活起來。
- 全部的圖示（例如打勾、地圖圖釘）都是「向量圖（SVG）」，保證放大 100 倍也絕對不模糊！

## 🚀 如何在自己電腦上打開這個網站？

這是一個純靜態網頁，不需要安裝任何複雜的伺服器環境或程式語言！

1. 把這個資料夾下載到您的電腦裡。
2. 找到 `index.html` 這個檔案，用您平常上網的瀏覽器（如 Chrome, Edge 或 Safari）直接點開兩下。
3. 恭喜！您就可以開始體驗這個網站了。
4. 別忘了點擊 `showcase.html`，裡面有更多白話文的網站巧思介紹喔！
